# .impeccable.md — Workflow Console

## Design Context

### Users
自用运维工具，使用者是项目创建者本人。日常在 Mac/浏览器中打开，快速查看 OpenClaw 定时任务状态、触发任务、管理 Hooks 和浏览社交采集内容。使用频率较高，以读为主、偶尔触发操作。不需要向他人演示，专注个人效率。

### Brand Personality
**三个词：精准、克制、自主**
不是产品，是工具。不追求华丽，追求"用起来顺"。像一个配置好的命令行 Dashboard——知道你要什么，安静地把信息呈现出来。

### Aesthetic Direction
- **主题**：深色系，保留现有方向
- **参考气质**：Linear Issue tracker、Raycast、Vercel Dashboard 的克制与精准——信息密度适中，有设计感但不表演
- **反参考**：不要 glassmorphism、不要发光霓虹边框、不要 cyan/purple 渐变、不要圆角卡片堆满屏幕
- **字体**：选有个性的等宽或科技感衬线，避免 Inter/Roboto
- **色彩**：深色背景带轻微暖色调或冷灰调（不要纯黑），1-2 个精准的强调色，不要彩虹

### Design Principles
1. **信息优先**：每个像素都为信息服务，装饰只在强化结构时才存在
2. **操作清晰**：主操作一眼可见，次要操作退后但可发现
3. **状态可信**：运行状态、错误、成功必须视觉上可区分且准确，不要模糊
4. **克制的个性**：有一两处记得住的设计细节，但整体不喧宾夺主
5. **密度适中**：不要太稀（浪费屏幕），不要太挤（难以扫读）
